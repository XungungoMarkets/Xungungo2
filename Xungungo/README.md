# Xungungo

Proyecto inicial para una app desktop con PySide6 + QML + LightweightCharts.

## Requisitos

- Python 3.11+

## Instalación

```bash
pip install -r requirements.txt
```

## Ejecutar

```bash
python run.py
```

## Notas

- La primera sección disponible es **Ticker**.
- Use un símbolo como `BTC-USD`, `AAPL` o `SPY`.
